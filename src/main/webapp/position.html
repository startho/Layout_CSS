position 기법(p.280)

포지션은 웹문서에서 이미지나 글씨등을 원하는 위치에 자유롭게 둘때 사용합니다.

position 속성은 자주 사용하지만, 그만큼 실수하기도 쉬운 부분이므로 꼼꼼하게 살펴보세요

position 종류(p.282)

  • static : 문서의 흐름에 맞게 배치(default), block은 block처럼, inline은 inline 처럼
  • relative : 위칫값(left,right,bottom,top)을 지정할수 있다는 점을 빼곤 static과 같다.
  • absolute : relative값을 사용한 상위 요소를 기준으로 위치를 지정해 배치합니다.
  • fixed : 브라우저 창(=모니터)을 기준으로 위치를 지정해 배치합니다.
  • sticky : static + absolute 방식으로 동작하는 새로운 위치 종류

position의 위치를 지정하는 속성

  • left : 기준위치와 해당 요소 사이의 왼쪽으로 얼마나 떨어져 있는지
  • right
  • bottom
  • top

position의 static은 위치지정 속성을 쓰더라도 적용되지 않습니다
